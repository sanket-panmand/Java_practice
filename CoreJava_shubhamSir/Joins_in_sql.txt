SQL Joins......
=================================================================================

    -we used sql joins to combine data or rows from two or more tables
	based on common fields between them. 
	-Types of Joins are
	
	   1.Inner join - All the matching rows from both tables
	   2.Left Join  - All rows from left table and matching rows from right table
	   3.Right Join - All rows from right table and matching rows from left table
	   4.Full Join 
	      - It is a combination of result of LEFT JOIN and RIGHT JOIN.
		  - Here in ther output contains all the rows from both the tables.
