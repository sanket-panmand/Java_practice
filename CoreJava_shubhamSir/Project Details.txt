Project Description/ Summary

User Driven Registration - New User
This is a user driver new customer registration of bank,
where customer can register itself to the bank for new account opening.
This application of bank is used by user where he can put basic details, address details,
organisation details and required details for registrtion in bank.
He can initiate KYC also through this application.
